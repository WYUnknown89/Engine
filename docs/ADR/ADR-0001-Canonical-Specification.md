# ADR-0001 Canonical Specification

## Status

Accepted

## Context

The project maintains Markdown and PDF documentation.

## Decision

The Markdown specification is the canonical v1.1 source. The PDF is a
secondary human-readable artifact generated from the Markdown and cannot
override it.

## Consequences

All future edits begin in Markdown. The visible version in both artifacts must
match before a documentation milestone is complete.
